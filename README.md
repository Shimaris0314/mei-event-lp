# ME:I Event Landing Page

## 概要
ME:Iの架空のイベント告知LPをコーディングしたポートフォリオです。HTML/CSS/JavaScriptを使用し、レスポンシブ対応とハンバーガーメニューを実装。

## 使用技術
- HTML5: セマンティックなマークアップ
- CSS3: Flexbox、レスポンシブデザイン、アニメーション
- JavaScript: ハンバーガーメニューの開閉

## 工夫点
- レスポンシブ対応でPC/スマホ両対応
- クリーンなコードとSEOを意識
- ファン目線のUI/UXを考慮したデザイン
